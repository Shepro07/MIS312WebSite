Project02 — Milestone 1: Sensor Familiarization (Run Only)

Timing: Weeks 1–3
Tools: Raspberry Pi 3 B+ + GrovePi+ (identical wiring), RealVNC, GrovePi Python examples
Development Tool: RealVNC + terminal (no editing required yet)
Submission: OneNote Artifact (PDF)

Objective

Build confidence by running provided GrovePi sensor scripts and observing output without changing wiring.

Student Tasks

Task 1 — Connect to your Raspberry Pi

Open RealVNC Viewer → connect to your assigned Pi → locate the GrovePi sample code folder (shown in class).

Task 2 — Run at least TWO sensor scripts

Run at least two different sensors (light, sound, button, ultrasonic distance, potentiometer).
Observe output for 30+ seconds each.

Task 3 — Record what you observed

For each sensor:

  • What does it measure?
  • What values did you observe?
  • What physical action changed the values?

Task 4 — AI explanation (required)

Ask:

“Explain what this GrovePi sensor script is doing at a high level (beginner-friendly).”

OneNote Artifact Requirements

  • Screenshot of each sensor script running (output visible)
  • Short written explanation for each sensor
  • AI prompt + summary
  • Reflection: easiest sensor vs most confusing sensor and why
