Timing: Weeks 13–15
Tools: Raspberry Pi + GrovePi+
Development Tool: VS Code SSH
Submission: OneNote Artifact (PDF) + full project folder to D2L

Objective

Combine at least two sensors into one program and present results as a clear “dashboard” output.

Student Tasks

  1. Choose two sensors.
  2. Combine into one program that reads both and reports status clearly.
  3. Make output understandable to non-programmers (labels, OK/WARNING/ALERT).
  4. AI review + rubric check (required): ask for a 0–4 readiness score and one improvement; implement it.

VS Code SSH Exercise — Multi-Sensor Dashboard as .py Project (Required)

Task 5 — Build your dashboard as a small project folder on the Pi

On the Pi (via VS Code SSH), create:

  • ~/MIS312/project02/final/ Inside, include:
  • dashboard.py
  • (optional) helpers.py for functions

Task 6 — Run and demonstrate reliability

Run:

      python3 dashboard.py

Task 7 — Troubleshooting (required)

Demonstrate one reliability improvement:

  • Example: handle a temporary bad reading without crashing
  • Example: ensure your program exits cleanly with Ctrl+C

Document with screenshots and 2–3 sentences explaining what changed.
