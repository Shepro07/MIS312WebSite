Project02 — Milestone 4: Data Logging + Simple Visualization

Timing: Weeks 10–12
Tools: Raspberry Pi + GrovePi+, laptop JupyterLab
Development Tool: VS Code SSH (Pi) + JupyterLab (analysis)
Submission: OneNote Artifact (PDF) + project files to D2L

Objective

Log sensor data to CSV and create a simple summary + plot.

Student Tasks

  1. Write sensor values to CSV with timestamp/value/status; log 50+ rows.
  2. Transfer CSV to laptop.
  3. In JupyterLab: load CSV, compute min/max/avg, create one line plot.
  4. AI summary (required): ask for a business-friendly summary; write 3–5 sentences.

VS Code SSH Exercise — CSV Logging on the Pi (Required)

Task 5 — Create the logger script on the Pi

  1. Connect to the Pi with VS Code Remote - SSH.
  2. Create: milestone4_logger.py
  3. Your script must write a CSV file on the Pi (50+ rows).

Run:

      python3 milestone4_logger.py

Task 6 — Troubleshooting (required)

Simulate a “bad path” problem:

  • Change the output filename to a folder that doesn’t exist → observe error
  • Fix by creating the folder or using a correct path

Task 7 — Transfer the CSV to your laptop for JupyterLab analysis

Use the method shown in class (SCP, shared folder, or browser download).

Add to OneNote Artifact: proof of the CSV on the Pi + proof it opened on your laptop.

OneNote Artifact Requirements

  • Evidence CSV created (screenshot/file)
  • Notebook snippet with summary + plot
  • AI prompt + summary
  • Reflection: what you learned from the plot
